# Contact Details Scraper (Sample)

Sample inputs for the Apify Contact Details Scraper tool.

## Files
- `a.csv`: Example CSV with a `Website` column listing domains.
- `keys.json`: Example format for Apify API tokens (replace with your real keys).

## CSV Format
```
Website
example.com
koldify.io
apify.com
```

## keys.json Format
```
[
  "apify_api_TOKEN_1",
  "apify_api_TOKEN_2"
]
```

## Usage
Select `a.csv` and `keys.json` from this folder via the Contact Details Scraper UI card, choose your output folder, batch size, output mode, and optional departments filter.
